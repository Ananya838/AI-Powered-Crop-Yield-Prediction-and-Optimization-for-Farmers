# Demo Script (7-8 minutes)

1. Landing Story (45 sec)
- Explain farmer pain points: uncertain yield, high input cost, unpredictable pest outbreaks.

2. Farmer Login + Dashboard (1 min)
- Register/login farmer.
- Show predicted yield, water savings, and active pest risk.

3. Yield Prediction Workflow (2 min)
- Enter crop, district, weather, and soil values.
- Show expected yield + confidence + trend graph + alternate crop suggestion.

4. Optimization Plan (1.5 min)
- Generate irrigation schedule, fertilizer dosage, sowing/harvest window.
- Highlight projected productivity gain (>10%).

5. Pest Alert + Weather Insight (1 min)
- Show risk classification and action advisory.
- Display weather trend chart for decision support.

6. Admin Analytics (1 min)
- Show district-level productivity and adoption analytics.
- Explain targeted intervention using heatmap data.

7. Close (30 sec)
- Summarize measurable impact and scalability for nationwide use.
